# Shortcutter

**Shortcutter** — это лёгкая Windows-утилита для управления медиаплеерами (YouTube, Spotify и др.) с помощью глобальных хоткеев.

Программа работает в фоновом режиме, отображается в системном трее и позволяет управлять воспроизведением даже тогда, когда браузер или плеер находятся в фоне (например во время игры).

---

# Основные возможности

• глобальные хоткеи для управления медиа  
• управление YouTube, Spotify и другими плеерами через системные media keys  
• всплывающие уведомления (OSD overlay)  
• иконка в системном трее  
• гибкая настройка хоткеев  
• включение или отключение отдельных функций  
• конфигурация сохраняется в пользовательской директории  

---

# Поддерживаемые действия

По умолчанию доступны следующие действия:

| Функция | Хоткей |
|--------|--------|
| Toggle hotkeys | F6 |
| Play / Pause | Ё |
| Next track | F8 |
| Play / Pause (media key) | F9 |
| Mute | F10 |
| Volume down | F11 |
| Volume up | F12 |

Все хоткеи можно изменить в настройках программы.

---

# Настройки

Окно настроек содержит две вкладки.

## General

Настройки отображения уведомлений:

• длительность уведомления  
• время анимации появления  
• время анимации исчезновения  
• позиция уведомления на экране  

Также можно включить или выключить работу хоткеев.

---

## Hotkeys & Features

Позволяет:

• изменить хоткеи  
• включать или отключать отдельные функции  

Например можно отключить:

• mute  
• изменение громкости  
• переключение треков  

---

# Системный трей

Программа работает через иконку в системном трее.

Меню содержит:

```
Enabled
Hotkeys
Settings
Exit
```

Иконка меняет цвет:

🟢 зелёная — хоткеи включены  
🔴 красная — хоткеи отключены  

---

# Конфигурация

Файл настроек хранится в:

```
%APPDATA%\Shortcutter\config.json
```

Пример структуры конфигурации:

```json
{
  "enabled": true,

  "osd": {
    "hold_ms": 3500,
    "fade_in_ms": 140,
    "fade_out_ms": 240,
    "offset_x": 34,
    "offset_y": 34
  },

  "hotkeys": {
    "toggle": "f6",
    "playpause_ru": "ё",
    "next": "f8",
    "playpause": "f9",
    "mute": "f10",
    "voldown": "f11",
    "volup": "f12"
  },

  "features": {
    "playpause": true,
    "next": true,
    "mute": true,
    "voldown": true,
    "volup": true
  }
}
```

# Архитектура проекта

```
shortcutter/
│
├── main.py
├── hotkeys.py
├── osd.py
├── media.py
├── settings.py
├── config.py
│
├── requirements.txt
├── README.md
└── .gitignore
```

Описание модулей:

**main.py**  
точка входа приложения, создание tray и запуск сервисов

**hotkeys.py**  
обработка глобальных хоткеев через `pynput`

**osd.py**  
overlay-уведомления поверх всех окон

**media.py**  
отправка Windows media key событий через WinAPI

**config.py**  
загрузка и сохранение конфигурации

**settings.py**  
окно настроек

# Используемые технологии

Python  
PySide6 (Qt GUI)  
pynput (global hotkeys)  
Windows WinAPI (media key events)

# Установка

Скачайте последнюю версию программы со страницы **Releases**.

Запустите:

```
shortcutter.exe
```

Программа автоматически создаст конфигурацию в папке:

```
%APPDATA%\Shortcutter
```

---

# Запуск из исходников

```
git clone https://github.com/username/shortcutter
cd shortcutter
pip install -r requirements.txt
python main.py
```
